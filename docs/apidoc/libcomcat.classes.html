toDict(catalog=None, get_all_magnitudes=False, get_tensors='preferred', get_moment_supplement=False, get_focals='preferred')[source]¶

Return known origin, focal mechanism, and moment tensor information for a DetailEvent.

Parameters:
  • catalog – Retrieve the primary event information (time,lat,lon…) from the catalog given. If no source for this information exists, an AttributeError will be raised.
  • get_all_magnitudes – Boolean indicating whether all known magnitudes for this event should be returned. NOTE: The ComCat phase-data product’s QuakeML file will be downloaded and parsed, which takes extra time.
  • get_tensors – String option of ‘none’, ‘preferred’, or ‘all’.
  • get_moment_supplement – Boolean indicating whether derived origin and double-couple/source time information should be extracted (when available.)
  • get_focals – String option of ‘none’, ‘preferred’, or ‘all’.
Returns:

OrderedDict with the same fields as returned by SummaryEvent.toDict(), preferred moment tensor and focal mechanism data. If all magnitudes are requested, then those will be returned as well. Generally speaking, the number and name of the fields will vary by what data is available.

getContentName(regexp)[source]¶

Get the shortest filename matching input regular expression.

For example, if the shakemap product has contents called grid.xml and grid.xml.zip, and the input regexp is grid.xml, then grid.xml will be matched.

Parameters:regexp – Regular expression to use to search for matching contents.
Returns:Shortest file name to match input regexp, or None if no matches found.
